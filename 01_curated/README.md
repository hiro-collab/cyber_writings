# 01_curated

`00_fragments` の断片を、主題や時系列ごとに整理したノートを置く。

ここでは、元の意味を保ちながら、重複、時系列、論点を見える形にする。

## 整理の単位

- 症状の進行
- 認知機能の変化
- 記憶、注意、言語、感情、睡眠
- サイバー、情報環境、自己認識
- 社会、医療、制度、仕事
- 原稿化できそうな論点

## 目的

`02_manuscript` に直接進む前に、素材の地図を作る。

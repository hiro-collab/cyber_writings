# 00_fragments

ChatGPT履歴、メモ、日記、会話ログなどから、あとで使う可能性のある断片を置く。

ここでは完成度よりも、出典と状態を残すことを優先する。

## 推奨ファイル名

- `YYYY-MM-DD_short-title.md`
- `unknown-date_short-title.md`
- `chatgpt_YYYY-MM-DD_topic.md`

## 入れるもの

- 元の発言やメモの抜粋
- その断片が出てきた日付や文脈
- 公開可能かどうか
- 事実、感覚、解釈、仮説の区別
- 後で調べるべき点

## 注意

ChatGPTエクスポートの `conversations.json` などをそのまま置かない。必要な部分だけを抜き出す。

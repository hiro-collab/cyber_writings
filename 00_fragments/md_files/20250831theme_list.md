## この文書の扱い

この文書は、`project/00_raw/source_docs/20250831theme_list.md` をそのまま清書するためのものではない。  
raw 側の一覧は、`chat011` における assistant 生成のテーマ分類結果であり、user の原文ではない。  
そのため、この修正文案では assistant が足した一般論や論文化の枠組みを外し、source-backed な主題だけを残す。

また、current user feedback により、`思想・実装・祈り` / `思想・祈り・実装` の三層構造は誤情報混入の疑いが強い。  
`祈り` は user 定義自体はあるものの、今回の文書整理では混乱を招きやすいため採用しない。  
`陰陽` は、0/1 や ON/OFF のような二状態の区別を指す限定語としてのみ扱う。  
したがって、この文書では `祈り` を軸にした整理や、属性論を伴う `陰陽` の語り方を採用しない。
